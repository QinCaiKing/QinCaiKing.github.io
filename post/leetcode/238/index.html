238.除自身以外数组的乘积

1.思路与代码

  • 除自身以外的乘积，最容易想到的方式就是双重循环，然后逐次计算。但是这样做的时间复杂度达到了**O(n^2)**，没有满足题目要求。
  • 那么需要在常数级的循环内计算出结果，就需要考虑最终结果的组成方式。如果一次遍历，那么能算出的乘积相对于每一个元素而言，顶多是到当前元素之前的所有元素的乘积，即如下例所示
    • nums:[1,2,3,4,5]
    • result:[11,12,123,1234,1234*5]
    • 这样，就比较容易的获得了，对于每个元素的前缀元素的乘积。除去自己以外的元素乘积 = 前缀元素乘积*后缀元素乘积。那么所以，两次循环即可，一次正序算前缀，一次倒序算后缀。
