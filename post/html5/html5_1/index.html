HTML5学习笔记(一)

1.HTML部分 标签

  • HTML作为标签化语言，其是由标签组成的，通过不同的标签组成一颗DOM树来展示页面、这样就导致标签间存在着两种关系，并列关系与包含关系。下面的示例中，h1标签与p标签是并列关系，html标签与body标签则是包含关系。另外下面的示例中，标签是成对存在的，开始的标签中只存在有标签关键词，而结束的标签中在关键词前有一个/符号。

  • 除了成对出现的标签以外，还存在有单独出现的标签。例如img标签等。写法则需要在关键词后加/符号。

  • HTML标签与其内部的内容构成了HTML元素，例如h1标签与其内部的文字就构成了一个元素。标签代表元素展示的形式，而内容则代表元素展示什么。另外在标签上也可以存在有属性。下面这部分代码中的href属性就代表着a标签的指向。

2.HTML文本标签

  • 标题标签 <h1></h1>--<h6></h6>六种标题标签，在html中为加粗，加大的单行文本，随着标签内数字的增大，标题逐渐减小，标题文字放在双标签之间。
  • 段落标签<p></p>将段落放在双标签之间，段落间会有空行。
  • 换行标签<br/>在HTML文本中会屏蔽换行与空格，将此标签写在需要换行的地方即可，单标签。、
  • 水平线标签
    标签在 HTML 页面中创建水平线。即在书写标签的前后之间展示一条水平分割线。

3.HTML文本格式化标签

  • <strong></strong>或<b></b>粗体
  • <em></em>或<i></i>斜体
  • <del></del>或<s></s>删除线
  • <ins></ins> 或 <u></u>下划线
  • 正常文本<sub></sub>下标
  • 正常文本<sup></sup>上标
  • <code></code>代码文本

4.链接标签

  • 图像链接标签<img src="" alt="" title="" width="" height=""> 。其中src属性表示图片路径，属于必填属性，选择路径下的图片将会显示；alt属性为替换文本，表示当图像不可以显示时，替换在图片位置区域的文字；title为提示文字，即鼠标放置在图片上显示的文字信息；height属性与width属性为图像的高度与宽度。
  • 超链接标签<a href="" target=""></a> 其中href属性表示指定连接的url地址或者锚点地址；target属性表示打开位置拥有_self和_blank两种可填写数据，前者为在本页打开，后者为在新页打开。 当需要使用a标签做发送邮件的功能时，需要使用mailto关键字，具体使用方法为<a href="mailto:邮件接收地址?subject=这是邮件的主题&body=这是邮件的内容" rel="nofollow">发送邮件</a>
  • 路径可以使用绝对路径与相对路径，绝对路径是机器上资源位置的全路径，相对路径表示与当前文件相对的路径，同一文件夹不需要写前缀直接调用即可；上一级文件夹需要使用../来充当前缀；下一级文件夹需要使用/前缀，**/前为文件夹，/后为文件**。

5.头部标签

  • <head>元素包含了所有的头部标签元素。在<head>元素中你可以插入脚本（scripts）, 样式文件（CSS），及各种meta信息。

  • 标题标签<title></title>，这个标签定义了浏览器工具栏的标题；显示在搜索引擎结果页面的标题，显示在收藏夹中的标题。作为标题不单单可以显示文本，还可以显示logo图片，在头部中添加如下标签即可实现在标题中显示图片<link rel="shortcut icon" href="图片url">。

  • <base href="",target=""/>标签定义了一个基本的链接地址/目标，其作为此文档中所有的链接标签的默认链接。使用方式与a标签相同。此时如果在a标签或其他标签中书写相对路径，指向的链接地址将不再是当前的域，而是此标签中定义的地址。

  • <link ref="" type="" href="">标签定义文档与外部资源之间的关系，通常用于链接到样式表。书写方式为<link rel="stylesheet" type="text/css" href="样式表链接">

  • <style> 标签定义了HTML文档的样式文件引用地址，也可以直接在标签中添加样式来渲染HTML文档。

  • meta标签描述了一些基本的元数据。<meta>标签提供了元数据.元数据也不显示在页面上，但会被浏览器解析。META 元素通常用于指定网页的描述，关键词，文件的最后修改时间，作者，和其他元数据。元数据可以使用于浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他Web服务。

  • <script>标签用于加载脚本文件，如： JavaScript。

6.表格标签

  • 表格<table></table>定义表格的标签，最外层
  • 表格行<tr></tr>定义表格行的标签，一行，嵌套在表中
  • 表头单元格<th></th>定义表头单元格的标签，会加粗并居中显示，一般放置在第一行，嵌套在行中
  • 表格单元格<td></td>定义单元格的标签，嵌套在行中
  • 头部区域<thead></thead>表示表格的表头区域，标签内部必须有<tr></tr>标签，一般放在第一行，嵌套在表中
  • 表体区域<tbody></tbody>表示表格的表体区域，嵌套在表中
  • 跨行合并单元格rowspans属性，值为行数，写在上方单元格，删除下方单元格
  • 跨列合并单元格colspan属性，值为列数，写在左侧单元格，删除右侧单元格

7.列表标签

  • 无序列表<ul></ul>表示无序列表，列表项之间没有先后顺序，标签中只能嵌套<li></li>标签，不可以嵌套其他标签；<li></li>表示列表项，可以嵌套其他标签。无序列表可以拥有样式，使用CSS完成。
  • 有序列表<ol></ol>表示有序列表，列表项之间存在先后顺序，会自动编号，标签中只能嵌套<li></li>标签，不可以嵌套其他标签；<li></li>表示列表项，可以嵌套其他标签。有序列表可以拥有样式，使用CSS完成。
  • 自定义列表<dl></dl>自定义列表常用于对术语或名词进行解释和描述，定义列表的列表项前没有任何项目符号。<dt></dt>为定义项目或者名字，<dd></dd>围绕项目进行解释或说明。标签中只能嵌套<dt></dt>或<dd></dd>标签，不可以嵌套其他标签；<dt></dt>与<dd></dd>表示列表项，可以嵌套其他标签。自定义列表可以拥有样式，使用CSS完成。

8.盒子标签

  • <div></div>一行只能放置一个div盒子容器，为块元素
  • <span></span>一行可以放置多个span盒子容器，din内部可以包含span，为行内元素

9.表单标签

  • 表单域<form></form>标签内包裹的都属于表单域
  • 表单元素<input>标签表示输入框，type属性表示此输入框内部的属性，可以以很多形式表现，输入框，选择框等,value属性为默认值，name属性为此元素的名，maxlength属性表示输入字段中的字符最大长度，checked属性表示选择的勾选。
  • 表单元素type可取值button定义可点击按钮（多数情况下，通过JS启动脚本）；checkbox定义复选框；file定义输入字段和”浏览按钮”，供文件上传；hidden定义隐藏的输入字段；image定义图像形式的提交按钮；password定义密码字段，此字段中的字符输入后会被掩码；radio定义单选按钮；reset定义重置按钮，重置按钮会清除表单中的所有数据；submit定义提交按钮，提交按钮会把表单数据发送到服务器；text定义单行的输入字段，用户可在其中输入文本，默认宽度为20个字符。
  • 表单标注标签<label></label>会使得此标签内部与input进行联合，点击标注标签相当于与点击input标签，要求for属性与input的id属性相同。
  • 选项标签<select></select>表示下拉列表，至少包含一对<option></option>标签，<option></option>表示选项集中的选项。当选项中的selected属性为selected时，表示当前项为默认选中项，不写的话默认为第一个选项。
  • 文本域标签<textarea></textarea>多行文本，cols属性为一行的文字数，rows属性为行数

10.内联框架

  • 内联框架可以在html页面中嵌套一个其他的页面，<iframe src="URL" width="" height="" frameborder=""></iframe>，其中src代表链接目标地址，width与height代表宽度与高度，frameborder代表是否显示边框，值为0时候不显示边框。还可以搭配a标签，完成iframe内展示页面。

11.脚本标签

  • 在script标签中可以直接书写操作页面的脚本语句，例如想要在页面打开时，弹出一个警告信息，可以写为。在这个页面中，页面会先弹出一个警告，警告中为123。这就是脚本中语句的作用，可以对页面进行一些逻辑或动作操作。

    当页面不支持脚本，或浏览器禁止脚本时，可以使用noscript标签来展示内容。此时页面会显示标签内的文字，并且不会执行脚本语句。
